・引継ぎ書の見方
1.ドライブでginrei57→部門ファイル→情報宣伝のところから{引継ぎ}というファイルを丸ごとダウンロードする.
2.ダウンロード出来たら,windowsに標準でついているエクスプローラーを開く.
3.開いたら左のタブからダウンロードを選択する.
4.上のほうに{引継ぎ (ダウンロードした日付) (よくわからん数字)}の圧縮された(zip)ファイルがあるので、そのファイル上で右クリックする.
5.右クリックしたら{すべて展開}という項目があるのでそれをクリックする.
6.なんか出てくるので{完了時に展開されたファイルを表示する}というチェックボックスにチェックを入れる.
7.そのあと右下の{展開}をクリックする.
8.展開が終わると,展開されたファイルが表示されるので,その中にある{引継ぎ}というファイルを左クリックで開く
9.なかに{body.html}というファイルがあるので,Edgeやchromeなどで開くと引継ぎ書が見られます.
※{引継ぎ}の中にあるファイル(body.html, style.css ,image)は,絶対に移動させない.(見れなくなります.)
　{引継ぎ}のファイル自体は、動かしてもよい
